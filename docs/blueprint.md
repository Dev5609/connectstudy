# **App Name**: ConnectStudy

## Core Features:

- Virtual Study Rooms: Users can create private or public study rooms, invite friends via a unique code or link, and view live focus timers for all participants within the room.
- Integrated Video & Voice Chat: Enable real-time, low-latency video and voice communication directly within study rooms, with controls to toggle camera and microphone status for individual users.
- In-Room Messaging & Sharing: Real-time chat functionality supporting text messages, image uploads, and document sharing via drag-and-drop within each study room, ensuring a focused communication channel.
- Versatile Focus Timer Modes: Offers three distinct timer modes: an infinite timer starting from zero, a custom session timer (e.g., 25, 40, 90 mins) with a 1-second soft beep on completion, and a full Pomodoro system with customizable focus and break durations.
- Personalized Focus Analytics: Tracks and displays daily, weekly, monthly, and total lifetime focus time, calculates average focus time per day, and records study session streaks, providing insights into productivity.

## Style Guidelines:

- A sophisticated black and white color scheme for a minimalist and distraction-free experience. Primary action color: Pure Black (#000000) for strong contrast and interactive elements. Background color: A clean, soft White (#FFFFFF) to ensure readability and a spacious feel. Accent color: A subtle Dark Gray (#333333) for secondary elements and subtle distinctions, providing depth without introducing color.
- Primary font: 'Inter' (sans-serif) for all text elements, chosen for its high readability, modern, and neutral appearance, which ensures a clean and objective interface across headlines and body text.
- Minimalist and clean line icons with a uniform stroke weight. Icons should be clear and immediately understandable, aligning with the distraction-free and modern aesthetic. Favor simple, intuitive glyphs over elaborate designs.
- Desktop-first design featuring a clean, uncluttered interface. Information hierarchy is established through strategic use of negative space and card-based layouts. Content should be centrally focused to minimize visual clutter, with navigation and auxiliary features thoughtfully integrated into sidebars or minimalist top bars.
- Subtle and purposeful animations and transitions. Focus on smooth, understated movements for interactive elements (e.g., button hovers, state changes, timer progress) that provide positive user feedback without becoming a source of distraction.
